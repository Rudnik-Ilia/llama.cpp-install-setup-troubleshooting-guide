# Useful Links & Resources

This section compiles links to the most helpful resources encountered while building and using Llama.cpp.

## 🔗 Official & Core Documentation
*   **[Repository README (Quick Start)](https://raw.githubusercontent.com/ggml-org/llama.cpp/master/README.md)**: The primary quick start guide from the official repo.
*   **[Server README (Raw)](https://raw.githubusercontent.com/ggml-org/llama.cpp/master/tools/server/README.md)**: Detailed documentation for the server tools.
*   **[llama-cli Manpage](https://manpages.debian.org/unstable/llama.cpp-tools/llama-cli.1.en.html)**: Official man page for the command-line interface.
*   **[llama-server Manpage](https://manpages.debian.org/unstable/llama.cpp-tools/llama-server.1.en.html)**: Official man page for the server.
*   **[llama-bench Manpage](https://manpages.debian.org/unstable/llama.cpp-tools/llama-bench.1.en.html)**: Official man page for the benchmarking tool.

*   **[Batch vs. uBatch](https://github.com/ggml-org/llama.cpp/discussions/6328)**: Comparison between batch processing modes.
*   **[Community "Ultimate Guide" (Reddit)](https://www.reddit.com/r/LocalLLaMA/comments/1h2hioi/ive_made_an_ultimate_guide_about_building_and/)**: A highly-recommended, comprehensive guide from the community.

*   **[Blog Guide (Steelph0enix Dev)](https://blog.steelph0enix.dev/posts/llama-cpp-guide/)**: A newer or parallel guide on the developer's blog.
